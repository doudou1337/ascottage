Check-in / Check-out

Arrivée (Check-in)

À partir de 16h00

Procédure :

  • Récupération des clés dans la boîte à clés sécurisée
  • Code d'accès envoyé par message avant votre arrivée
  • Visite libre du logement
  • Guide d'accueil disponible sur place

Départ (Check-out)

Avant 11h00

Procédure :

  • Laisser les clés dans la boîte à clés
  • Fermer toutes les fenêtres
  • Éteindre les lumières et appareils
  • Déposer les serviettes dans la salle de bain

Règles de la Maison

Respect du Logement

  • Traiter le logement avec soin comme si c'était le vôtre
  • Ne pas fumer à l'intérieur
  • Pas d'animaux de compagnie
  • Signaler immédiatement tout dommage ou problème
  • Utiliser les équipements selon les instructions

Nombre d'Invités

  • Maximum 2 personnes
  • Pas de fêtes ou d'événements
  • Visites d'invités avec accord préalable
  • Respecter la tranquillité des voisins

Nuisances Sonores

  • Calme respecté après 22h00
  • Pas de musique forte
  • Conversations téléphoniques discrètes
  • Respect du voisinage en toutes circonstances

Propreté

  • Maintenir la propreté générale
  • Faire la vaisselle après utilisation
  • Sortir les poubelles si nécessaire
  • Laisser le logement dans l'état de votre arrivée

Sécurité

  • Verrouiller portes et fenêtres en partant
  • Ne pas laisser les clés à l'extérieur
  • Signaler toute personne suspecte
  • Respecter les consignes de sécurité incendie

Tri Sélectif

  • Respecter le tri des déchets
  • Poubelle jaune : recyclables
  • Poubelle verte : déchets ménagers
  • Compost disponible pour les déchets organiques
